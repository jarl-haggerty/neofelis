How to install Blast+

If you are using a Windows, Linux, or Solaris computer you can download the version of Blast+ for your platform here here. For Macs however Blast+ must be built from source. This is not that hard but the build process can take a very long time.

Download the source code and put it in a safe place. Now open "Applications/Utilities/Terminal" and enter these commands

  1. cd {where did you save the download?}/c++
  2. ./configure --without-debug --without-pcre --with-mt --with-flat-makefile --with-universal --with-ncbi-public --with-build-root=Release --without-ccache --without-caution --without-makefile-auto-update --with-projects=scripts/projects/blast/project.lst --with-internal
  3. make

When you tell the pipeline the location of Blast+ select {where did you save the download?} /c++/Release

How to install a database

Go to the list of blast databases and search for a series of files called {your database}.{number}.tar.gz. Now, choose a folder to store your database in and extract the contents of each of the archives for your database into this folder. When you specify the location of your databases for the pipeline you're going to specify the previous folder.

How to install Genemark

Go to the download page of the Genemark website, check GeneMarkS, your operating system, enter your information, and agree to the license. If you use Windows then you'll have to download cygwin and the Linux 32 version of GaneMarkS.

How to install Transterm

Transterm is only available as source code, available here. Now open a terminal and enter these commands

  1. cd {where did you save the download?}
  2. make

When you tell the pipeline the location of Transterm select {where did you save the download?}
